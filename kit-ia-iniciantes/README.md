# Kit IA para Iniciantes

Produto digital completo para quem quer aprender a usar inteligência artificial do zero.

## Estrutura

```
kit-ia-iniciantes/
├── ebook/                    # E-book completo (7 capítulos)
│   ├── 01-introducao.md
│   ├── 02-o-que-e-ia.md
│   ├── 03-ferramentas.md
│   ├── 04-prompt-engineering.md
│   ├── 05-casos-praticos.md
│   ├── 06-erros-comuns.md
│   └── 07-proximos-passos.md
├── templates/                # 50+ prompts prontos
│   ├── prompts-para-trabalho.md
│   ├── prompts-para-estudo.md
│   ├── prompts-para-criacao.md
│   └── prompts-para-vida.md
├── cheatsheet/               # Referência rápida de 1 página
│   └── cheatsheet-ia-iniciantes.md
├── landing-page/             # Página de vendas
│   └── index.html
└── assets/                   # Imagens e recursos
```

## Como usar

1. **E-book**: Leia do início ao fim (formato .md, pode ser convertido para PDF)
2. **Templates**: Copie e cole os prompts na ferramenta de IA de sua escolha
3. **Cheatsheet**: Consulte rapidamente durante o uso diário
4. **Landing page**: Hospede em qualquer servidor para vender o kit

# CHEATSHEET IA para Iniciantes

## Estrutura do Prompt Perfeito

```
CONTEXTO + TAREFA + FORMATO + TOM
```

## Exemplo Prático

"**Sou dono de uma loja de roupas.** [CONTEXTO]
**Crie 5 posts para Instagram** [TAREFA]
**cada um com título + texto curto + hashtags** [FORMATO]
**Tom: casual e divertido** [TOM]"

## Técnicas Rápidas

| Técnica | Como fazer | Exemplo |
|---|---|---|
| **Persona** | Peça pra IA assumir um papel | "Atue como um personal trainer..." |
| **Refinamento** | Peça ajustes | "Mais curto", "Mais formal" |
| **Exemplos** | Dê um modelo | "Como este exemplo: [...]" |
| **Cadeia de ideias** | Peça passo a passo | "Explique passo a passo..." |

## 10 Prompts Essenciais

```
1. Resuma: "Resuma [texto] em 5 bullet points"
2. Explique: "Explique [conceito] como se eu tivesse 12 anos"
3. Crie: "Crie [X] ideias de [tema]"
4. Melhore: "Melhore este texto: [texto]"
5. Traduza: "Traduza para [idioma] mantendo o tom original"
6. Compare: "Compare [A] e [B] em formato de tabela"
7. Planeje: "Crie um plano de [X] dias/semanas para [objetivo]"
8. Analise: "Analise [texto/dados] e destaque os pontos principais"
9. Converta: "Transforme este texto em [formato] (tabela/lista)"  
10. Formate: "Reescreva em formato de [artigo/poste/email]"
```

## Erros Comuns

❌ Prompt muito curto → ✅ Dê contexto
❌ Aceitar resposta sem revisar → ✅ Você é o editor
❌ Não refinar → ✅ Prompting é iterativo
❌ Confiar em fatos → ✅ Verifique informações críticas

## Regra de Ouro

> IA é o motor. **Você é o piloto.**

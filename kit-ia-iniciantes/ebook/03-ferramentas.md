# Capítulo 2: Principais Ferramentas de IA

## Para texto (escrita, análise, conversa)

| Ferramenta | Ideal para | Gratuito? |
|---|---|---|
| **ChatGPT** | Conversas, escrita criativa, código | Sim (versão limitada) |
| **Claude** | Textos longos, análise profunda, documentos | Sim (versão limitada) |
| **Gemini** | Integração com Google, busca online | Sim |
| **DeepSeek** | Código e raciocínio lógico | Sim |

## Para imagens

| Ferramenta | Ideal para | Gratuito? |
|---|---|---|
| **Canva IA** | Design, redes sociais, apresentações | Sim |
| **Midjourney** | Imagens artísticas e profissionais | Pago |
| **DALL-E** | Imagens criativas a partir de texto | Pago |

## Para produtividade

| Ferramenta | Ideal para | Gratuito? |
|---|---|---|
| **Notion IA** | Escrita e organização | Sim (teste) |
| **Perplexity** | Pesquisas com fontes | Sim |
| **Gamma** | Apresentações com IA | Sim (teste) |

## Como escolher a ferramenta certa

- **Precisa escrever um texto longo?** → Claude
- **Precisa de uma resposta rápida?** → ChatGPT
- **Precisa pesquisar algo atual?** → Gemini ou Perplexity
- **Precisa criar uma imagem?** → Canva IA (iniciantes) ou Midjourney (profissional)
- **Precisa analisar um documento?** → Claude (aceita arquivos grandes)

> Dica: não precisa escolher apenas uma. Use a ferramenta certa para cada tarefa.

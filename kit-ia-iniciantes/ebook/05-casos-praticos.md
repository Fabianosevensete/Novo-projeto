# Capítulo 4: Casos Práticos para Usar Imediatamente

## No Trabalho

### Escrever e-mails mais rápido
```
Prompt: "Escreva um e-mail profissional para [situação].
Use tom cordial e direto. Máximo de 150 palavras."
```

### Analisar documentos longos
Jogue o PDF no Claude e peça:
```
Resuma este documento em 5 bullet points.
Depois, liste os 3 pontos mais importantes para tomar decisão.
```

### Criar apresentações
```
Prompt: "Crie uma estrutura de 10 slides para uma apresentação
sobre [tema]. Para cada slide, dê o título e 3 bullet points."
```

## Nos Estudos

### Explicar conceitos complexos
```
Prompt: "Explique [conceito] como se eu tivesse 12 anos de idade.
Use analogias do dia a dia."
```

### Criar flashcards
```
Prompt: "Crie 10 flashcards de estudo sobre [tema].
Cada flashcard: pergunta de um lado, resposta do outro."
```

### Resumir artigos e livros
```
Prompt: "Resuma o texto abaixo em 3 parágrafos.
Destaque os principais argumentos e conclusões."

[cole o texto aqui]
```

## Na Vida Pessoal

### Planejamento de refeições
```
Prompt: "Crie um plano de refeições para 7 dias.
Preciso de opções rápidas (menos de 30 min), que não usem
ingredientes difíceis de encontrar."
```

### Roteiros de viagem
```
Prompt: "Crie um roteiro de 3 dias em [cidade] para [perfil].
Inclua opções de café da manhã, almoço e jantar,
com atrações gratuitas e pagas."
```

### Aprender algo novo
```
Prompt: "Quero aprender [habilidade] do zero.
Crie um plano de estudos de 30 dias, com uma tarefa por dia."
```

## Para Criar Conteúdo

### Posts para redes sociais
```
Prompt: "Crie 5 posts para Instagram sobre [tema].
Cada post: título chamativo + texto de 5 linhas + 3 hashtags.
Tom: casual e direto."
```

### Roteiros de vídeo
```
Prompt: "Crie um roteiro de vídeo de 3 minutos sobre [tema].
Estrutura: gancho inicial (15s) + desenvolvimento (2min) + CTA (15s)."
```

### Legendas para YouTube/Reels/TikTok
```
Prompt: "Escreva uma legenda envolvente para um vídeo sobre [tema].
Inclua: gancho, desenvolvimento, chamada para ação e hashtags."
```

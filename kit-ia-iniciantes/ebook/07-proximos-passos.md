# Capítulo 6: Próximos Passos

## Como continuar evoluindo

Você já tem o básico para começar. O segredo agora é prática consistente.

### Semana 1: Exploração
- Use uma ferramenta de IA diferente a cada dia
- Teste os prompts deste guia
- Anote o que funcionou e o que não funcionou

### Semana 2: Aplicação
- Escolha UMA área (trabalho, estudos, criação)
- Use IA para uma tarefa real todos os dias
- Refine seus prompts até a resposta ficar boa

### Semana 3: Integração
- Incorpore IA nos seus fluxos de trabalho
- Crie seus próprios templates de prompt
- Compartilhe com alguém e peça feedback

### Semana 4: Domínio
- Explore técnicas avançadas (personas, correntes de pensamento)
- Combine ferramentas (use IA + outras ferramentas)
- Ensine alguém — ensinar é a melhor forma de aprender

## Glossário Rápido

| Termo | Significado |
|---|---|
| **IA Generativa** | IA que cria conteúdo novo |
| **Prompt** | Instrução que você dá para a IA |
| **Prompt Engineering** | Técnica de criar prompts eficientes |
| **Alucinação** | Quando a IA inventa informações falsas |
| **LLM** | Modelo de linguagem (o "cérebro" da IA de texto) |
| **Token** | Pedaço de texto que a IA processa (aproximadamente ¾ de uma palavra) |
| **Fine-tuning** | Treinar uma IA com dados específicos |
| **API** | Forma de conectar a IA com outros sistemas |

## Para se manter atualizado

- A área de IA muda rápido. O que funciona hoje pode mudar amanhã.
- Teste novas ferramentas quando aparecerem.
- O princípio mais importante não muda: **bons prompts = bons resultados**.

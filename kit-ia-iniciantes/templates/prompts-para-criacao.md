# Prompts Prontos para CRIAÇÃO DE CONTEÚDO

---

## Post para Instagram
```
Crie um post para Instagram sobre [tema].
Estrutura:
- Título chamativo (máximo 15 palavras)
- Texto de 5-8 linhas (tom casual e direto)
- 5 hashtags relevantes
- Chamada para ação no final
```

## Roteiro de Vídeo Curto (Reels/TikTok/Shorts)
```
Crie um roteiro para vídeo de até 60 segundos sobre [tema].
Estrutura:
- GANCHO (primeiros 3 segundos): frase que prende atenção
- DESENVOLVIMENTO: 3 pontos principais (15s cada)
- CTA: chamada para ação final (5s)
Tom: dinâmico e conversacional
```

## Roteiro de Vídeo para YouTube
```
Crie um roteiro para vídeo de YouTube sobre [tema].
Duração: [X] minutos.
Estrutura:
- INTRODUÇÃO: gancho + o que será abordado
- DESENVOLVIMENTO: divida em 3-4 blocos com transições
- CONCLUSÃO: resumo + CTA (inscreva-se, comente, compartilhe)
- Inclua sugestões de B-roll em cada bloco
```

## Artigo de Blog
```
Escreva um artigo de blog sobre [tema].
Extensão: 800-1000 palavras.
Inclua:
- Título SEO
- Subtítulos (H2, H3)
- Introdução engajadora
- Desenvolvimento com exemplos práticos
- Conclusão com CTA
- Palavra-chave principal: [palavra-chave]
```

## Legenda para YouTube
```
Escreva uma legenda envolvente para um vídeo intitulado "[título]".
Inclua:
- Gancho escrito (primeira linha chamativa)
- Resumo do conteúdo
- Timestamps se aplicável
- Links relevantes
- CTA (inscreva-se, veja o próximo)
- 5-8 hashtags
```

## Copy para Anúncio
```
Crie um texto de anúncio para [plataforma] sobre [produto/serviço].
Público-alvo: [descrever público]
Diferenciais: [lista de diferenciais]
Tom: [casual / profissional / urgente]
Inclua: headline, corpo do texto, CTA.
```

# Prompts Prontos para ESTUDO

---

## Explicar Conceito (Modo Infantil)
```
Explique [conceito] como se eu tivesse 12 anos.
Use analogias do mundo real.
Exemplo: se for explicar nuvem computacional, compare com um condomínio.
```

## Plano de Estudos Personalizado
```
Quero aprender [habilidade/tema] do zero.
Crie um plano de estudos de 30 dias.
Cada dia: tópico, tempo estimado, exercício prático.
Considere que tenho [X] horas por dia para estudar.
```

## Flashcards
```
Crie 15 flashcards de estudo sobre [tema].
Cada flashcard:
PERGUNTA: [pergunta]
RESPOSTA: [resposta em 1-2 linhas]
```

## Mapa Mental
```
Crie um mapa mental sobre [tema].
Tema central: [tema]
Ramos principais: liste os 5-7 conceitos principais
Sub-ramos: para cada conceito, dê 2-3 detalhes importantes
```

## Resumo para Revisão
```
Resuma o texto abaixo em 20% do tamanho original.
Mantenha os argumentos principais e dados importantes.
Use bullet points para facilitar a revisão.

[Texto aqui]
```

## Perguntas de Estudo
```
Crie 10 perguntas de múltipla escolha sobre [tema].
Cada pergunta: 4 alternativas (A, B, C, D) + resposta correta.
Nível: [básico / intermediário / avançado]
```

# Prompts Prontos para TRABALHO

---

## E-mail Profissional
```
Assunto: [assunto]

Escreva um e-mail profissional sobre [assunto] para [destinatário].
Tom: cordial e direto.
Extensão: máximo 150 palavras.
Inclua: saudação, contexto, solicitação principal e despedida.
```

## Relatório com Dados
```
Analise os dados abaixo e crie um relatório executivo de 1 página.
Inclua: principais insights, tendências identificadas e recomendações.
Formato: tópicos com dados de suporte.

[Dados aqui]
```

## Resumo de Reunião
```
Resuma esta reunião em:
1. Decisões tomadas
2. Próximos passos (com responsáveis)
3. Pontos em aberto
Formato: bullet points.

[Transcrição ou notas da reunião]
```

## Feedback para Funcionário
```
Crie um feedback construtivo para um funcionário que [situação].
Use a técnica SBI (Situação, Comportamento, Impacto).
Tom: positivo e profissional.
Termine com uma sugestão de melhoria.
```

## Proposta Comercial
```
Crie uma proposta comercial para [serviço/produto] para [cliente].
Inclua: problema identificado, solução proposta, benefícios, investimento.
Tom: persuasivo mas profissional.
Extensão: máximo 400 palavras.
```

## Avaliação de Currículo
```
Analise este currículo para a vaga de [cargo].
Destaque: pontos fortes, gaps, e sugestões de melhoria.
Ofereça sugestões específicas para cada tópico.

[Currículo aqui]
```

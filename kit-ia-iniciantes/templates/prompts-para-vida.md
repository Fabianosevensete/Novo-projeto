# Prompts Prontos para VIDA PESSOAL

---

## Plano de Refeições Semanal
```
Crie um plano de refeições para 7 dias (café, almoço, jantar).
Restrições: [restrições alimentares]
Preferências: [gostos pessoais]
Tempo máximo por refeição: [X] minutos
Inclua lista de compras ao final.
```

## Rotina de Exercícios
```
Crie uma rotina de exercícios para [dias por semana] dias.
Nível: [iniciante / intermediário / avançado]
Equipamento: [academia / casa sem equipamentos / mínimo]
Duração: [X] minutos por sessão
Objetivo: [perder peso / ganhar massa / manter saúde]
```

## Roteiro de Viagem
```
Crie um roteiro de [X] dias em [cidade/destino].
Perfil: [casal / família / solo / amigos]
Orçamento: [econômico / médio / alto]
Inclua: atrações, restaurantes, transporte, dicas locais.
Distribua as atividades por período (manhã/tarde/noite).
```

## Planejamento Financeiro Pessoal
```
Me ajude a criar um planejamento financeiro mensal.
Renda: R$ [valor]
Despesas fixas: [listar]
Objetivos: [curto prazo / longo prazo]
Crie um orçamento em categorias com valores sugeridos.
```

## Carta Pessoal
```
Escreva uma carta para [destinatário] sobre [assunto/ocasião].
Tom: [sincero / emocionado / formal / casual]
Inclua: memória especial, agradecimento, mensagem principal.
```

## Diário Guiado (Reflexão)
```
Atue como um guia de journaling. Faça 5 perguntas profundas
para eu refletir sobre [tema da semana / momento de vida].
As perguntas devem me ajudar a ganhar clareza e tomar ação.
```
